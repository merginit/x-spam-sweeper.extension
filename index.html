X Spam Sweeper

A lightweight Chrome extension to batch report and block spam accounts in X (Twitter) Messages.

Get Started View Repository

Clean your inbox in seconds.

Tired of manually dealing with dozens of spam message requests? X Spam Sweeper lets you view, select, and batch-process spam accounts directly from a convenient popup saving you time and clicks.

  • ✓ One-click Sweep: Report → Block → Delete
  • ✓ AI-powered Spam Detection (Gemini Nano)
  • ✓ Native X Design Language

Powerful Capabilities

Designed to be privacy-first, fast, and effective.

Smart Scoring

Privacy-first filtering with a 0-30 point scale. Analyzes URLs, keywords, and AI risk factors locally.

Multi-Select

Select individual accounts or use "Select All" to choose multiple spammers for batch processing.

Link Resolution

Securely resolves t.co shortlinks to identify actual destination domains before you ever click them.

Visual Risk Indicators

Instantly identify suspicious messages with color-coded badges.

HIGH RISK

Score ≥ 20. Likely blatant spam or malicious links.

MEDIUM RISK

Score ≥ 10. Suspicious keywords or patterns found.

LOW RISK

Score ≥ 3. Mild indicators found, use caution.

Installation

Get up and running in minutes.

1

Clone Node

Clone the repository:
git clone https://github.com/merginit/x-spam-sweeper.extension.git

2

Extensions Page

Open Chrome and navigate to chrome://extensions.
Enable "Developer mode".

3

Load Unpacked

Click "Load unpacked" and select the cloned repository folder.

Privacy & Security

No analytics, tracking, or external requests. All processing happens locally in your browser. The extension only reads data from the X messages page; it does not store or transmit any information externally.
